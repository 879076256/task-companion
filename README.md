# Task Companion

Task Companion 是一个处于规划阶段的 Obsidian 插件项目。本轮只建立安全开发环境、官方插件骨架、质量命令与规划文档，尚未实现正式功能。

## 目录

- `task-companion/`：基于 Obsidian 官方 Sample Plugin 的插件工程。
- `test-vault/`：仅供后续测试使用的隔离人造 Vault。
- `docs/`：产品与工程规划文档。
- `reference/`：参考来源与材料说明。

## 当前安全状态

- 不包含真实任务或真实 Vault 数据。
- 不连接、读取或修改任何正式 Obsidian Vault。
- 不包含计时器、后台轮询、遥测或网络请求。
- 插件入口仅提供空的加载与卸载生命周期。

详见 `docs/PROJECT_STATUS.md` 与 `docs/CURRENT_PLAN.md`。
