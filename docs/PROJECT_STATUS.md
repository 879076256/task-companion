## Phase 8 完成内容

- 母任务复盘保存并关闭后可靠打开“沉淀经验模板”，支持新建、更新已有模板或暂不沉淀。
- 模板包含一层子任务顺序、历史平均执行次数与投入、常见阻塞、完成前检查和下次沿用的经验；两个可选经验字段提供直观说明与示例。
- 执行准备提供模板建议和预览，用户点击采用后才添加缺失子任务；已有同名子任务不会重复创建。
- 模板使用 schemaVersion 1 的追加式 JSONL，读取时隔离损坏行并按版本折叠。
- 插件实例公开版本化 API `1.0.0`，覆盖当前目标、计时、会话、复盘、模板、界面入口和事件订阅。
- 提供 task-selected、timer-started/paused/resumed/finished、session-saved、subtask-created/completed、task-completed、review-created/completed 共 11 个事件。
- 受控扩展位于 `TaskCompanion/Scripts/`，只允许声明式 JSON；默认停用、显式启用、最小权限、错误脱敏、失败回退或停用。
- 复盘组件显示“今日已完成”，按本地 `completedAt` 筛选并按 `taskId + subtaskId` 去重；点击同形指标进入今日队列，悬停文字变紫且显示手型指针。
- “打开完整复盘队列”主页按钮已移除；命令面板仍可打开完整复盘队列。

## 安全与范围

- 不执行用户提供的任意 JavaScript，不使用 `eval` 或 `Function`。
- 不提供网络、Shell、任意文件访问、任意 Vault 写入或危险权限。
- 扩展监听器和声明式扩展失败不得中断任务、计时、会话或复盘的持久化。
- 不实现脚本市场、AI 自动修改、正式主页安装、多层子任务、依赖图或甘特图。
