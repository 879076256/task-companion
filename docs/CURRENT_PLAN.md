## Subtask 数据流

选择稳定 taskId → 读取该任务追加事件 → 折叠为一层 Subtask 当前状态 → 添加/改名/排序/完成/取消/返工/设为下一步 → 追加新事件 → 计时或快速推进选择母任务或一个活动子任务 → ExecutionSession 保存可空 subtaskId → 汇总母任务直接投入、各子任务投入和完成数。
