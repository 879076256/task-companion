# 当前计划

## 本轮范围

只完成 Phase 6：任务完成分流、待复盘队列、自动统计和可读复盘。基线为人工验收通过的 `v0.5.0-breakdown`。

## 完成分流

- 无会话且无子任务档案：安全勾选原任务，直接退出，不进入复盘。
- 存在会话或子任务档案：冻结自动统计，完成原任务并加入待复盘队列。
- 存在活动子任务：完成前必须明确选择返回继续、取消剩余或保留记录并完成。
- 复盘失败：原任务保持完成，完整复盘留在 `data.json` 待写队列，命令重试。

## 复盘统计

任务跨度、实际执行天数、会话数、有效/暂停时间、提前结束次数、初始与执行中新增子任务、完成与取消数、最长耗时步骤、最后进展和未完成子任务。
